# Changelog

## [1.1.0] - 2026-02-16

### Added
- ปุ่มลบในหน้า profile
- หน้าสำหรับเลือกข้อมูลที่จะลบ
- ระบบ Cron Job สำหร้บทำ Hard Delete ข้อมูลในฐานข้อมูลอัตโนมัติ 
- ฟีเจอร์ลบข้อมูลของผู้ใช้ที่ผู้ใช้สามารถเลือกได้ เช่น routes, booking, vehicle หรือเลือกลบข้อมูล Account ทั้งหมด
- ฟีเจอร์ส่งข้อมูลที่ผู้ใช้ลบผ่านทาง Email

### Changed
- ปรับเพิ่มเงื่อนไขการ query ที่เกี่ยวข้องกับตาราง routes, booking, vehicle, user
